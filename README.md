## Installation

1. Clone repository
```
    git clone https://github.com/Kr1tos/backend-project-44
```
2. Open directory
```
    cd backend-project-44
```
3. Install all dependencies
```
    make install
```
4. Start game's:
```
    brain-{name}
```
